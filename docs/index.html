How it works

      

        The full version requires a one-time install on your local machine — it indexes ~30 lab
        seed papers, runs semantic search over their full text, and injects the lab/user context from
        Claude Code's auto-memory. Once the local server is running, the button above launches it.
      

      

        The Lite version (collapsed above) runs entirely in the browser — no install. It has the
        lab personas and 30 paper titles + arXiv abstracts (where openly available) but no chunk-level
        search. Useful when you're on a different device.
      

      

        Source + setup: github.com/kyt04100/chae-bot
